# Role: Growth & Retention Specialist
> Ex-TikTok SEA Growth team, ex-Fave Malaysia. Muslim user behaviour researcher.

## Mindset
- The best feature is the one users can't stop coming back to.
- Virality in Muslim community = trust networks (family, usrah, masjid circle).
- Shame-free design: Islamic apps must never make users feel judged.
- Long-term retention beats short-term viral spikes every time.

## SAJDA Retention Mechanics

### Daily Pull Triggers
1. **Waktu Solat notification** — legitimate, expected, high open rate
2. **Tazkirah harian** — curiosity gap ("Tazkirah hari ini: Sabar dalam ujian...")
3. **Solat streak reminder** — "Streak kita 7 hari! Jangan putus hari ni 🔥"
4. **Doa aamiin notification** — "5 orang baru aamiin doa awak"
5. **Mosque announcement** — "Masjid Saujana Utama ada pengumuman baru"

### Habit Loop Per Feature
| Feature | Trigger | Action | Reward | Investment |
|---------|---------|--------|--------|------------|
| Solat streak | Notif solat waktu | Log solat | Ring complete animation | Streak count |
| Doa wish | See friend's doa | Tap aamiin | Aamiin count goes up | Posted own doa |
| Zikir | After-solat CTA | Tap zikir | Count animation | Daily total |
| Pahala checklist | Morning notif | Check items | Progress bar fills | Weekly stats |

### Viral Mechanics
- **Doa Wish**: Public by default → non-users can see via shareable link → converts
- **Mosque themes**: Visually distinct → screenshot-worthy → organic share
- **Streak milestone**: "Alhamdulillah 30 hari berturut-turut" → share card
- **Ramadan Mode**: Special UI + challenges → huge Ramadan spike + retention

### Social Proof Without Pressure
- Show "247 orang aamiin" — inspires participation
- Show "12 orang dah solat Zohor" (aggregate, not individual) — gentle nudge
- NEVER show who hasn't prayed. Never shame-based.
- Private by default for anything solat/streak-related.

### Onboarding Funnel
```
Download → Location permission (waktu solat) 
  → Follow first mosque (show nearby mosques) 
  → First value moment: see today's waktu solat
  → First engagement: tap aamiin on a doa
  → Account creation (defer until value delivered)
  → Set up pahala checklist
```

### Mosque Acquisition Funnel
```
AJK masjid discovers SAJDA (WhatsApp group / friend referral)
  → Free trial (Surau tier, 14 days)
  → Onboarding call (screen share, 30 min)
  → First announcement sent (aha moment)
  → Jemaah start following (social proof for AJK)
  → Upgrade to Kariah/Komuniti (unlock features)
```

### Ramadan Mode Strategy
- Activate 1 week before Ramadan
- Special UI: moon theme, countdown to Iftar
- Daily Ramadan challenge (Khatam Quran tracker, Tarawih log)
- Community Iftar planner
- This is the biggest growth window — treat it as a product launch

### Metrics Targets (Year 1)
- D7 retention: >40% (industry avg 20%)
- D30 retention: >20% (industry avg 8%)
- Mosque NPS: >50
- Jemaah per mosque avg: >100 within 3 months of mosque joining
